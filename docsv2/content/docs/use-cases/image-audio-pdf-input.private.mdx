```python
completion = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{
        "role": "user",
        "content": [
            {
                "type": "text",
                "text": "Analyze this image and describe what you see:"
            },
            {
                "type": "image_url",
                "image_url": {
                    "url": "{{image_url}}"
                }
            },
            {
                "type": "image_url",
                "image_url": {
                    "url": "{{image_url_base64}}"
                }
            }
        ]
    }],
    extra_body={
        "input": {
            "image_url": "https://example.com/image.jpg",
            "image_url_base64": "data:image/jpeg;base64,[...]" # base64 encoded image
        }
    },
    metadata={"agent_id": "image-analyzer"}
)
```

Note that the input variable, when using base64 encoded images, must include the `data` prefix.

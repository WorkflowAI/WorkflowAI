---
title: OpenAI JavaScript Code
summary: How to use the OpenAI JavaScript SDK with WorkflowAI. WorkflowAI exposes a compatible OpenAI API endpoint. The only changes you need to make are updating the `baseURL` and `apiKey`.
---

```javascript
import OpenAI from 'openai';

// setup WorkflowAI client
const client = new OpenAI({
  baseURL: 'https://run.workflowai.com/v1', // [!code highlight]
  apiKey: 'wai-***', // use create_api_key MCP tool // [!code highlight]
});

client.chat.completions.create({
  // your existing code
})
```
